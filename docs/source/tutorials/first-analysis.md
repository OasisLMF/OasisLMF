# Run your first analysis

```{admonition} Proof-of-concept placeholder
:class: note

This is a placeholder tutorial demonstrating the Diátaxis "Tutorials" mode. A
real version would walk a newcomer end-to-end through running a small analysis
with an example model (e.g. PiWind), from installation to inspecting the ORD
output — ideally as an executable page so every command shown is actually run at
build time.
```

A finished tutorial in this slot would:

1. Install OasisLMF (linking to {doc}`../installation`).
2. Fetch an example model and exposure set.
3. Run a ground-up + insured-loss analysis via the CLI.
4. Point at the {doc}`../reference/index` for the options used, and at
   {doc}`../explanation/financial-module` for *why* the insured-loss step does
   what it does.

The cross-links above illustrate the payoff of the Diátaxis split: a learner
follows the tutorial, then steps sideways into reference or explanation exactly
when they need depth.
